A simple paint application created for the purpose of learning about and implementing various design patterns.

## Built With

* Java
* JavaFX

## Design Patterns

The design patterns that were researched and used in this application.

### Creational

* Abstract Factory
* Prototype

### Structural

* Composite
* Facade

### Behavioral

* Subject-Observer
* Command
* Template method

### Other
* Model-View-Controller
